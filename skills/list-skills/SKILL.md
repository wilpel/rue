# list-skills

List all available Rue skills.

## Usage

```bash
node --import tsx/esm skills/list-skills/run.ts
```

## Output

Prints a list of all skills found in the skills directory, with their name and description (from SKILL.md).

## When to use

When the user asks "what skills do you have?" or "what can you do?" or you need to discover available skills.
