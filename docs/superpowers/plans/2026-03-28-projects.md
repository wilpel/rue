# Projects System Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a filesystem-based project management skill that lets Rue create workspaces, manage task boards, and auto-trigger agents for project work.

**Architecture:** A single skill at `skills/projects/` following the existing pattern (SKILL.md + run.ts CLI). Projects live at `~/.rue/workspace/projects/<name>/` with task files, agent instructions, docs, and a `work/` dir for actual files. Task creation emits an event file that the trigger system picks up to spawn agents.

**Tech Stack:** TypeScript CLI (run via `node --import tsx/esm`), filesystem for storage, YAML frontmatter in markdown task files.

# projects

Manage projects — organized workspaces with task boards, agent instructions, and documentation. Projects let Rue delegate sustained work to autonomous agents.

## Usage

```bash
# Create a new project
node --import tsx/esm skills/projects/run.ts create --name "my-api" --description "REST API for task management" --max-agents 2

# Create from a git repo (clones into work/)
node --import tsx/esm skills/projects/run.ts create --name "my-api" --git "https://github.com/user/repo.git"

# List all projects
node --import tsx/esm skills/projects/run.ts list

# Show project details (config, tasks, active agents)
node --import tsx/esm skills/projects/run.ts status --project "my-api"

# Add a task (creates task file, emits trigger event for agent spawning)
node --import tsx/esm skills/projects/run.ts add-task --project "my-api" --task "Set up Express with TypeScript"

# Update task status
node --import tsx/esm skills/projects/run.ts update-task --project "my-api" --task 1 --status in-progress --agent agent_abc
node --import tsx/esm skills/projects/run.ts update-task --project "my-api" --task 1 --status done

# Archive a project
node --import tsx/esm skills/projects/run.ts archive --project "my-api"
```

## Project structure

Each project lives at `~/.rue/workspace/projects/<name>/`:
- `PROJECT.md` — What this project is, goals, context. You write this after creating.
- `AGENTS.md` — Instructions for agents working on this project. You write this after creating.
- `config.json` — Metadata (name, description, maxAgents, status, tags)
- `docs/` — Project knowledge. Keep docs/notes.md updated with discoveries.
- `tasks/` — One file per task (e.g. `001-setup-express.md`) with YAML frontmatter
- `work/` — All working files (cloned repos, generated code). Agents operate here.

## Task file format

```markdown
---
id: 1
status: todo
created: 2026-03-28T21:10:00Z
agent: null
started: null
completed: null
---

# Task title

Task description here.
```

Status: `todo` → `in-progress` → `done` | `failed`

## When to use

- User asks for sustained work: "build me an API", "research X", "set up deployment"
- Work has multiple steps or spans multiple interactions
- Do NOT create projects for quick questions or single-shot tasks

## After creating a project

1. Write meaningful content into PROJECT.md (goals, context, approach)
2. Write AGENTS.md with project-specific agent instructions (code style, rules, workflow)
3. Add tasks with add-task — each task auto-triggers an agent
```

Create `skills/projects/run.ts`:
```typescript
#!/usr/bin/env tsx
import * as fs from "node:fs";
import * as path from "node:path";
import * as os from "node:os";
import { execSync } from "node:child_process";

const WORKSPACE = path.join(os.homedir(), ".rue", "workspace");
const PROJECTS_DIR = path.join(WORKSPACE, "projects");
const EVENTS_DIR = path.join(WORKSPACE, "events");

const args = process.argv.slice(2);
const command = args[0];

function getArg(name: string): string | undefined {
  const idx = args.indexOf(`--${name}`);
  return idx >= 0 && idx + 1 < args.length ? args[idx + 1] : undefined;
}

function ensureDirs(): void {
  fs.mkdirSync(PROJECTS_DIR, { recursive: true });
  fs.mkdirSync(EVENTS_DIR, { recursive: true });
}

function projectDir(name: string): string {
  return path.join(PROJECTS_DIR, name);
}

switch (command) {
  case "create": {
    ensureDirs();
    const name = getArg("name");
    const description = getArg("description") ?? "";
    const maxAgents = parseInt(getArg("max-agents") ?? "1", 10);
    const gitUrl = getArg("git");

    if (!name) {
      console.error("Usage: run.ts create --name <name> --description <desc> [--max-agents N] [--git <url>]");
      process.exit(1);
    }

    const dir = projectDir(name);
    if (fs.existsSync(dir)) {
      console.error(`Project "${name}" already exists at ${dir}`);
      process.exit(1);
    }

    // Create directory structure
    fs.mkdirSync(dir, { recursive: true });
    fs.mkdirSync(path.join(dir, "docs"), { recursive: true });
    fs.mkdirSync(path.join(dir, "tasks"), { recursive: true });
    fs.mkdirSync(path.join(dir, "work"), { recursive: true });

    // config.json
    const config = {
      name,
      created: new Date().toISOString(),
      description,
      maxAgents,
      status: "active",
      tags: [] as string[],
    };
    fs.writeFileSync(path.join(dir, "config.json"), JSON.stringify(config, null, 2));

    // Scaffold files with placeholders for Rue to fill in
    fs.writeFileSync(
      path.join(dir, "PROJECT.md"),
      `# ${name}\n\n${description}\n\n<!-- Rue: fill in project goals, context, and approach -->\n`,
    );
    fs.writeFileSync(
      path.join(dir, "AGENTS.md"),
      `# Agent Instructions\n\nYou are working on the ${name} project.\n\n<!-- Rue: fill in project-specific agent rules, code style, workflow -->\n\n## Working directory\nAll work happens in the work/ directory. Do not modify files outside work/.\n\n## When you pick up a task\n1. Update the task file: status to in-progress, add your agent ID and started timestamp\n2. Do the work in work/\n3. Update the task file: status to done, add completed timestamp\n4. Update docs/notes.md if you learned something relevant\n`,
    );
    fs.writeFileSync(path.join(dir, "docs", "notes.md"), `# Notes\n\nProject discoveries and decisions.\n`);

    // Clone git repo if provided
    if (gitUrl) {
      console.log(`Cloning ${gitUrl}...`);
      try {
        execSync(`git clone ${gitUrl} "${path.join(dir, "work", name)}"`, { stdio: "inherit" });
      } catch {
        console.error(`Failed to clone ${gitUrl}`);
      }
    }

    console.log(`Created project: ${name}`);
    console.log(`  Path: ${dir}`);
    console.log(`  Max agents: ${maxAgents}`);
    if (gitUrl) console.log(`  Cloned: ${gitUrl}`);
    console.log(`\nNext: Write content into PROJECT.md and AGENTS.md, then add tasks.`);
    break;
  }

  case "list": {
    ensureDirs();
    const entries = fs.readdirSync(PROJECTS_DIR, { withFileTypes: true });
    const projects = entries.filter((e) => e.isDirectory() && fs.existsSync(path.join(PROJECTS_DIR, e.name, "config.json")));

    if (projects.length === 0) {
      console.log("No projects.");
    } else {
      console.log(`${projects.length} project(s):\n`);
      for (const p of projects) {
        const config = JSON.parse(fs.readFileSync(path.join(PROJECTS_DIR, p.name, "config.json"), "utf-8"));
        const tasksDir = path.join(PROJECTS_DIR, p.name, "tasks");
        const taskFiles = fs.existsSync(tasksDir) ? fs.readdirSync(tasksDir).filter((f) => f.endsWith(".md")) : [];
        const todoCount = taskFiles.filter((f) => {
          const content = fs.readFileSync(path.join(tasksDir, f), "utf-8");
          return content.includes("status: todo");
        }).length;
        const inProgressCount = taskFiles.filter((f) => {
          const content = fs.readFileSync(path.join(tasksDir, f), "utf-8");
          return content.includes("status: in-progress");
        }).length;
        const doneCount = taskFiles.filter((f) => {
          const content = fs.readFileSync(path.join(tasksDir, f), "utf-8");
          return content.includes("status: done");
        }).length;
        console.log(`  ${config.name} [${config.status}]`);
        console.log(`    ${config.description}`);
        console.log(`    Tasks: ${todoCount} todo, ${inProgressCount} in-progress, ${doneCount} done`);
        console.log(`    Max agents: ${config.maxAgents}\n`);
      }
    }
    break;
  }

  case "status": {
    ensureDirs();
    const name = getArg("project");
    if (!name) { console.error("Usage: run.ts status --project <name>"); process.exit(1); }
    const dir = projectDir(name);
    if (!fs.existsSync(dir)) { console.error(`Project not found: ${name}`); process.exit(1); }

    const config = JSON.parse(fs.readFileSync(path.join(dir, "config.json"), "utf-8"));
    console.log(`Project: ${config.name} [${config.status}]`);
    console.log(`Description: ${config.description}`);
    console.log(`Max agents: ${config.maxAgents}`);
    console.log(`Created: ${config.created}\n`);

    const tasksDir = path.join(dir, "tasks");
    const taskFiles = fs.existsSync(tasksDir) ? fs.readdirSync(tasksDir).filter((f) => f.endsWith(".md")).sort() : [];
    if (taskFiles.length === 0) {
      console.log("No tasks.");
    } else {
      console.log("Tasks:");
      for (const f of taskFiles) {
        const content = fs.readFileSync(path.join(tasksDir, f), "utf-8");
        const statusMatch = content.match(/status:\s*(\S+)/);
        const agentMatch = content.match(/agent:\s*(\S+)/);
        const titleMatch = content.match(/^#\s+(.+)$/m);
        const status = statusMatch?.[1] ?? "unknown";
        const agent = agentMatch?.[1] !== "null" ? agentMatch?.[1] : null;
        const title = titleMatch?.[1] ?? f;
        const icon = status === "done" ? "x" : status === "in-progress" ? "~" : " ";
        console.log(`  [${icon}] ${title} (${status})${agent ? ` @${agent}` : ""}`);
      }
    }
    break;
  }

  case "add-task": {
    ensureDirs();
    const name = getArg("project");
    const task = getArg("task");
    if (!name || !task) { console.error("Usage: run.ts add-task --project <name> --task <description>"); process.exit(1); }
    const dir = projectDir(name);
    if (!fs.existsSync(dir)) { console.error(`Project not found: ${name}`); process.exit(1); }

    const tasksDir = path.join(dir, "tasks");
    fs.mkdirSync(tasksDir, { recursive: true });

    // Determine next task number
    const existing = fs.readdirSync(tasksDir).filter((f) => f.endsWith(".md"));
    const nextNum = existing.length + 1;
    const slug = task.toLowerCase().replace(/[^a-z0-9]+/g, "-").replace(/^-|-$/g, "").slice(0, 50);
    const filename = `${String(nextNum).padStart(3, "0")}-${slug}.md`;

    const taskContent = `---
id: ${nextNum}
status: todo
created: ${new Date().toISOString()}
agent: null
started: null
completed: null
---

# ${task}
`;

    fs.writeFileSync(path.join(tasksDir, filename), taskContent);

    // Emit trigger event
    const event = {
      type: "task-added",
      project: name,
      taskFile: `tasks/${filename}`,
      task,
      timestamp: Date.now(),
    };
    fs.writeFileSync(path.join(EVENTS_DIR, "task-added.json"), JSON.stringify(event, null, 2));

    console.log(`Added task #${nextNum}: ${task}`);
    console.log(`  File: tasks/${filename}`);
    console.log(`  Trigger event emitted.`);
    break;
  }

  case "update-task": {
    ensureDirs();
    const name = getArg("project");
    const taskNum = getArg("task");
    const status = getArg("status");
    const agent = getArg("agent");
    if (!name || !taskNum || !status) {
      console.error("Usage: run.ts update-task --project <name> --task <number> --status <status> [--agent <id>]");
      process.exit(1);
    }
    const dir = projectDir(name);
    if (!fs.existsSync(dir)) { console.error(`Project not found: ${name}`); process.exit(1); }

    const tasksDir = path.join(dir, "tasks");
    const taskFiles = fs.readdirSync(tasksDir).filter((f) => f.endsWith(".md")).sort();
    const taskFile = taskFiles.find((f) => f.startsWith(String(taskNum).padStart(3, "0")));
    if (!taskFile) { console.error(`Task #${taskNum} not found`); process.exit(1); }

    const filePath = path.join(tasksDir, taskFile);
    let content = fs.readFileSync(filePath, "utf-8");

    content = content.replace(/status:\s*\S+/, `status: ${status}`);
    if (agent) {
      content = content.replace(/agent:\s*\S+/, `agent: ${agent}`);
    }
    if (status === "in-progress") {
      content = content.replace(/started:\s*\S+/, `started: ${new Date().toISOString()}`);
    }
    if (status === "done" || status === "failed") {
      content = content.replace(/completed:\s*\S+/, `completed: ${new Date().toISOString()}`);
    }

    fs.writeFileSync(filePath, content);
    console.log(`Updated task #${taskNum}: ${status}${agent ? ` (agent: ${agent})` : ""}`);
    break;
  }

  case "archive": {
    ensureDirs();
    const name = getArg("project");
    if (!name) { console.error("Usage: run.ts archive --project <name>"); process.exit(1); }
    const dir = projectDir(name);
    if (!fs.existsSync(dir)) { console.error(`Project not found: ${name}`); process.exit(1); }

    const configPath = path.join(dir, "config.json");
    const config = JSON.parse(fs.readFileSync(configPath, "utf-8"));
    config.status = "archived";
    fs.writeFileSync(configPath, JSON.stringify(config, null, 2));
    console.log(`Archived project: ${name}`);
    break;
  }

  default:
    console.log("Usage: run.ts <create|list|status|add-task|update-task|archive> [options]");
    console.log("\nCommands:");
    console.log("  create      Create a new project");
    console.log("  list        List all projects");
    console.log("  status      Show project details");
    console.log("  add-task    Add a task to a project");
    console.log("  update-task Update task status");
    console.log("  archive     Archive a project");
}
```

## Projects

You manage projects in `~/.rue/workspace/projects/`. Use the projects skill to create and manage them.

### When to create a project
- User asks for sustained work: "build me an API", "research X", "set up a deployment pipeline"
- Work that has multiple steps or will take multiple interactions
- Anything that benefits from organized tasks, documentation, and agent delegation

### When NOT to create a project
- Quick questions: "what files are in this dir?", "explain this code"
- Single-shot tasks: "rename this variable", "fix this typo"
